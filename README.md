## 📖 项目概述

**Project Jupiter** 是一个硬核科幻共享宇宙，围绕"烛龙加速器"这一核心设定展开。采用**世界观-角色-故事**三层解耦架构，支持多条独立故事线。

- **Created:** 2026-02-12
- **Authors:** Rex Shen & Victoria (🐈‍⬛)
- **Status:** Active Development
- **Genre:** Hard Sci-Fi (Near Future to Far Future)
- **Scale:** Kardashev Type I → Type II Civilization

## 🌍 核心世界观

### 烛龙加速器 (The Zhulong)
- **类型：** 环绕太阳的巨型粒子加速器
- **位置：** 垂直于黄道面（形成"光环"）
- **目标：** 实现曲速引擎 (Alcubierre Drive)
- **代价：** 扭曲太阳系时空结构，形成"零曲率陷阱"

### 五幕式时代
1. **创世纪 (Genesis)** - 建造与博弈
2. **黄金时代 (Golden Age)** - 运转与停滞
3. **突破 (Breakthrough)** - FTL 成功与出走
4. **奇点陷阱 (Singularity Trap)** - 时空囚笼
5. **回归 (Return)** - 亿万年后

## 🚀 快速开始

### 创建新角色
1. 在 `characters/` 下创建 `{角色名}.md`
2. 填写基础信息、可选故事线
3. 在 README 的角色表中登记

### 创建新故事线
1. 在 `stories/` 下创建新文件夹 `{story_name}/`
2. 创建 `outline.md` 大纲
3. 逐步添加章节文件

### 扩展世界观
1. 在 `worldbuilding/` 相应子文件夹添加设定
2. 保持与现有设定的一致性
3. 更新本 README 的核心世界观摘要
