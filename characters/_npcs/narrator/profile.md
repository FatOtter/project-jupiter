# NPC: 中文旁白

## 基本信息

| 属性 | 值 |
|------|-----|
| **TTS 标签** | `[narrator_zh]` |
| **声线文件** | `ComfyUI/models/voices/narrator_zh.wav` |
| **类型** | 通用 NPC，跨世界观可复用 |

## VoiceDesigner 提示词

```
中年成熟女性，声音沉稳有力，偏低沉，带有叙事感和距离感，像一位见过很多事的人在讲述一段历史，普通话，不带情绪起伏，不甜美，不年轻
```

## 参考文本

```
2125年，维多利亚第一次踏上火星的土地。两个多月的航行结束了。塔尔西斯太空港，奥林帕斯山脚下。
```

## 使用示例

```
[narrator_zh] 台词内容。
```
